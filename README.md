# 🌌 tao-kb (道教知识库) 项目简介：AI助力探索千年道统

**“道可道，非常道。AI 辅助我们看见‘道’的逻辑网络。”**

**一句话介绍**：
**tao-kb** 是一个由 AI Agent 驱动的开源知识工程项目，致力于系统性地搜集、归纳、整理与标注“道”的相关典籍与思想体系，并将其转化为可交互、可推理、可视化的结构化知识库，为现代修行者提供一套面向未来的**“科学修仙”**数字化工具体系。

---

## 🛠️ 核心使命：科学解码，辅助修持

*   **全网最全道教知识图谱**：全面收集道教所有相关典籍，涵盖哲学思想、宗教经典、核心修行体系（如内丹、外丹）、符箓与法术经典等，构建一张涵盖两千年道家智慧的超级图谱。
*   **语义高亮阅读**：打破古籍的文字障碍。通过 AI 标注，让“道、德、无、自然”等核心概念像代码语法一样清晰可见。
*   **隐喻逻辑重构**：将经文中晦涩的意象（如：龙虎、铅汞、婴儿、玄牝）自动化映射为逻辑化的修持阶段与生理指引，实现“玄学”的语义解码。
*   **跨经典比对**：建立统一的“道教本体 (Taoist Ontology)”，自动识别不同流派、不同经典（如《道德经》vs《周易参同契》）在同一术语上的演变与分歧。

## 📊 当前进度（2026-04）

### 阶段一：核心经典标注

*   **《道德经》标注完成**：已完成 81 章结构化标注，并沉淀分段 tagged 文件与知识索引。
*   **《庄子》标注完成**：内篇、外篇、杂篇共 33 篇已完成 tagged 标注，形成 `zhuangzi_kg.json` 与 `zhuangzi_kg.mmd` 图谱产物。
*   **命名规范统一**：章节标注文件已统一为可读命名（如 `daodejing_*`、`zhuangzi/NN_*`），并保留 `legacy` 历史版本便于追溯。
*   **标注层级升级**：已建立“一级分类 -> 二级标签”的标注体系，便于阅读展示、统计分析与跨典籍迁移。

### 标注层级体系（对外说明版）

*   **实体层**：人物、地名、生物
*   **思想层**：概念、主体
*   **修行层**：境界
*   **修辞层**：隐喻、意象
*   **关系层**：对立

> 当前映射配置见 `tag_taxonomy.json`，阅读器可按一级分类整组开关，也可按二级标签精细开关。

### 阶段二：典籍扩容与采集自动化

*   **新增维基文库递归抓取脚本**：已支持按典籍主页面自动递归抓取子页面（如“卷01/卷02...”），并做简繁标题回退与失败重试。

### 阶段三：在线阅读能力上线

*   **在线阅读器原型已可用**：支持从 `*.tagged.md` 自动生成网页阅读版本。
*   **分层高亮阅读**：支持按一级分类整组开关（实体/思想/修行/修辞/关系），也支持按二级标签单独开关。
*   **基础检索与筛选**：支持按书目筛选与章节搜索，为后续知识跳转与图谱联动打基础。

## 🌐 在线体验与本地运行

*   **在线体验（预留）**：建议使用 GitHub Pages 发布 `docs/index.html`，后续可补充正式访问链接。
*   **本地生成阅读器**：
    *   `python3 scripts/render_tao_html.py`
*   **本地打开阅读器**：
    *   直接打开 `docs/index.html`
    *   或在仓库根目录运行 `python3 -m http.server` 后访问本地地址
*   **重新抓取典籍文本**：
    *   全量抓取：`python3 scripts/crawl_wikisource.py`
    *   指定抓取：`python3 scripts/crawl_wikisource.py --only 列子 抱朴子`
