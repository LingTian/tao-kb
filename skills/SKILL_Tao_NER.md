# SKILL_Tao_NER.md: 道教经典语义标注技能 (v1.0)

## 1. 任务目标
将《道德经》原文转化为结构化知识。通过在文本中插入特定标签，识别核心概念、隐喻、境界及辩证关系。

## 2. 标签体系 (Tags)
使用 `〖@类型:内容〗` 格式进行标注：

- **〖@概念:xxx〗**：核心哲学概念。
  - 示例：道、德、有、无、自然、无为。
- **〖@意象:xxx〗**：用于隐喻的自然物或具体事物。
  - 示例：水、谷、婴儿、牝、器。
- **〖@主体:xxx〗**：理想的人格或治理者。
  - 示例：圣人、侯王。
- **〖@境界:xxx〗**：描述状态或结果。
  - 示例：妙、徼、玄、静、朴。
- **〖@对立:xxx|yyy〗**：捕捉成对出现的辩证关系。
  - 示例：有|无、美|恶、长|短。

## 3. 标注原则
- **最小单元**：标注必须紧贴关键词。
- **语义优先**：如果一个词具有多重身份，选择其在当前语境下最核心的哲学身份。
- **覆盖率**：首阶段追求核心关键词的100%覆盖。

## 4. 输出示例
原文：上善若水。水善利万物而不争。
标注：上善若〖@意象:水〗。〖@意象:水〗善利万物而不争。
