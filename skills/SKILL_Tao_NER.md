# SKILL_Tao_NER.md: 道教经典语义标注技能 (v2.0 - 庄子扩展版)

## 1. 任务目标
将道家经典（《道德经》、《庄子》等）原文转化为结构化知识。通过在文本中插入特定标签，识别核心概念、人物、地理、生物及深层语义关系。

## 2. 标签体系 (Tags)
使用 `〖@类型:内容〗` 格式进行标注：

- **〖@概念:xxx〗**：核心哲学概念或术语。
  - 示例：道、无为、有待、六气、海运。
- **〖@隐喻:xxx〗**：具有深层象征意义的意象。
  - 示例：野马、尘埃、垂天之云。
- **〖@人物:xxx〗**：文中出现的历史、神话或寓言人物。
  - 示例：尧、舜、彭祖、宋荣子、列子。
- **〖@地名:xxx〗**：文中提及的地理位置（含寓言地名）。
  - 示例：北冥、南冥、姑射之山、汾阳。
- **〖@生物:xxx〗**：文中提及的动物、植物或特殊生命体。
  - 示例：鲲、鹏、蜩、学鸠、朝菌、蟪蛄、大椿。
- **〖@境界:xxx〗**：描述精神状态、生命层次或理想结果。
  - 示例：逍遥、天池、至人无己、圣人无名。
- **〖@对立:xxx|yyy〗**：捕捉成对出现的辩证关系或二元对比。
  - 示例：有|无、小知|大知、小年|大年、荣|辱。

## 3. 标注原则
- **最小单元**：标注必须紧贴关键词。
- **多重属性处理**：如果一个词既是生物又是隐喻（如“鲲”），在庄子语境下优先标注其具体的物理/寓言身份（如〖@生物:鲲〗），隐喻意义通过知识图谱后续关联。
- **保持连贯**：相同实词在文中多次出现时，标注需保持一致。

## 4. 输出示例
原文：北冥有鱼，其名为鲲。
标注：〖@地名:北冥〗有〖@生物:鱼〗，其名为〖@生物:鲲〗。
