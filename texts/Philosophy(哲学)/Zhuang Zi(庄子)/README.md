# 《庄子》 (Zhuangzi)

《庄子》是道家学派的经典著作，由战国中期的庄周及其后学所著。全书分为内篇、外篇、杂篇三个部分。

## 目录结构

*   **Inner Chapters (内篇)**: 庄子学说的核心，被普遍认为是庄子本人所作，包含《逍遥游》、《齐物论》、《大宗师》等7篇。
*   **Outer Chapters (外篇)**: 庄子后学对内篇思想的阐发和补充，包含《秋水》、《至乐》等15篇。
*   **Miscellaneous Chapters (杂篇)**: 杂糅了道家不同派别的文章，包含《天下》等11篇。
